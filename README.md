## ⚙️ Requirement

Sebelum install, pastikan kamu punya:

- ✅ **Sid Meier's Civilization VI** (versi terbaru via Steam)
- ✅ Windows 10 / 11
- ✅ DLC yang sesuai (kalau mau pakai terjemahan DLC tertentu)
- ❌ Tidak perlu mod launcher tambahan

---

## 🛠️ Cara Install (Mudah!)

> Tidak perlu aplikasi tambahan, cukup copy-paste file!

### Langkah 1 — Download
Klik tombol download di atas, pilih versi terbaru (`.zip`)

### Langkah 2 — Extract
Klik kanan file `.zip` → **Extract All**

### Langkah 3 — Copy File
Buka folder hasil extract, lalu copy semua isinya ke folder Mods Civ 6:

```
C:\Program Files (x86)\Steam\steamapps\common\Sid Meier's Civilization VI\paste disini
```

> 💡 Epic menyesuaikan ya

## ❓ FAQ

**Q: Apakah mod ini aman dipakai?**
A: Ya, 100% aman. Mod ini hanya mengubah teks tampilan, tidak mengubah gameplay sama sekali.

**Q: Apakah bisa dipakai untuk multiplayer?**
A: Bisa! Mod terjemahan hanya berpengaruh di tampilan lokal kamu, tidak mempengaruhi pemain lain.

**Q: Game saya crash setelah install, kenapa?**
A: Pastikan versi mod sesuai dengan versi game kamu. Coba disable mod lain dulu untuk mengecek konflik.

**Q: Kenapa ada teks yang masih Bahasa Inggris?**
A: Terjemahan masih dalam proses. DLC tertentu belum selesai diterjemahkan. Pantau update di halaman ini!

**Q: Bisa request terjemahan untuk DLC tertentu?**
A: Bisa! Buka Issues di GitHub dan request DLC yang kamu mau.

**Q: Apakah mod ini gratis?**
A: Ya, selamanya gratis! Tapi donasi sangat diapresiasi 💖

## 🐛 Menemukan Kesalahan Terjemahan?

Ada terjemahan yang aneh, salah, atau kurang tepat? Bantu kami perbaiki!

### Cara Report:
1. Buka halaman **[Issues](https://github.com/Kunosa-real/civ6-indonesian-translation/issues)**
2. Klik **New Issue**
3. Pilih template **"Kesalahan Terjemahan"**
4. Isi dengan informasi berikut:
   - 📍 Lokasi teks (menu apa, unit apa, dll)
   - ❌ Teks yang salah
   - ✅ Saran terjemahan yang benar
5. Klik **Submit**

> Setiap laporan sangat membantu meningkatkan kualitas terjemahan!
